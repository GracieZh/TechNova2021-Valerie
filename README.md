# TechNova2021-Valerie

Valerie is a web app chatbot that promotes mental wellness. 

## Install dependencies (frontend)

Use the package manager [npm](https://www.npmjs.com/package/npm) to install dependencies.

```bash
npm install
```

## Run (frontend)

```bash
expo start
```
